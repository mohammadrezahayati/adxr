
# ADXR (Nextjs 14 - admin panel)

A fully customizable and fully responsive admin panel created by tailwind + nextjs 14



# Support

This panel has RTL - LTR support that you can add every language you want.

```
{
     direction : RTL-LTR
     theme : Dark-Light
}
 ```
